## ğŸ® Usage

### Creating a Birth Chart

1. Open the app and tap "New Chart"
2. Enter birth details:
   - Date (DD/MM/YYYY)
   - Time (24-hour format)
   - Location (Latitude/Longitude)
   - Name (optional)
3. Select ayanamsa (Lahiri recommended for KP)
4. Tap "Generate Chart"

### Using the KP System

1. Generate a chart first
2. Navigate to the "KP Analysis" tab
3. View sub-lords for each cusp and planet
4. Check significators using the ABCD method
5. Analyze ruling planets for predictions

### Saving & Managing Charts

- Charts are automatically saved to local SQLite database
- Access saved charts from the "Library" screen
- Search by name or birth date
- Export charts as images
